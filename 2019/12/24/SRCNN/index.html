SRCNN

Posted on 2019-12-24 Edited on 2019-12-25 In SISR

Overview

SRCNNжҳҜж·ұеәҰз”ЁеңЁи¶…еҲҶиҫЁзҺҮйҮҚе»әдёҠзҡ„ејҖеұұд№ӢдҪңгҖӮе…¶жҢҮеҮәдј з»ҹзҡ„еҹәдәҺзЁҖз–Ҹзҹ©йҳөзҡ„ж–№жі•еҸҜд»ҘзңӢеҒҡдёҖдёӘж·ұеұӮзҘһз»ҸзҪ‘з»ңгҖӮSRCNNжҳҜдёҖдёӘиҪ»йҮҸзә§зҡ„зҪ‘з»ңпјҢеҸӘжңүдёүдёӘеҚ·з§ҜзҘһз»ҸзҪ‘з»ң(CNN)гҖӮдҪңиҖ…еңЁи®ӯз»ғж—¶е°ҶеӣҫзүҮиҪ¬еҢ–дёәYCrCbдёүдёӘйҖҡйҒ“пјҢеҸӘеҜ№YйҖҡйҒ“иҝӣиЎҢи®ӯз»ғгҖӮеӣ дёәдәәзңјеҜ№YйҖҡйҒ“жҜ”иҫғж•Ҹж„ҹгҖӮ

Model

ж•ҙдёӘSRCNNжЁЎеһӢеҸҜд»Ҙи§ЈйҮҠжҲҗдёүйғЁеҲҶ:

  • Path extraction and Representation.жҸҗеҸ–еӣҫеғҸPatchпјҢиҝӣиЎҢеҚ·з§ҜжҸҗеҸ–зү№еҫҒпјҢзұ»дјјдәҺзЁҖз–Ҹзј–з Ғдёӯзҡ„е°ҶеӣҫеғҸpatchжҳ е°„еҲ°дҪҺеҲҶиҫЁзҺҮеӯ—е…ёдёӯгҖӮ
  • Non-linear mapping.е°ҶдҪҺеҲҶиҫЁзҺҮзҡ„зү№еҫҒжҳ е°„дёәй«ҳеҲҶиҫЁзҺҮзү№еҫҒпјҢзұ»дјјдәҺеӯ—е…ёеӯҰд№ дёӯзҡ„жүҫеҲ°еӣҫеғҸpatchеҜ№еә”зҡ„й«ҳеҲҶиҫЁеӯ—е…ёгҖӮ
  • Reconstruction.ж №жҚ®й«ҳеҲҶиҫЁзҺҮзү№еҫҒиҝӣиЎҢеӣҫеғҸйҮҚе»әгҖӮзұ»дјјдәҺеӯ—е…ёеӯҰд№ дёӯзҡ„ж №жҚ®й«ҳеҲҶиҫЁзҺҮеӯ—е…ёиҝӣиЎҢеӣҫеғҸйҮҚе»әгҖӮ

йҰ–е…Ҳз”ЁеҸҢз«Ӣж–№жҸ’еҖјеҜ№йў„е…Ҳи®ҫи®ЎеҘҪзҡ„е°әеҜёеҜ№дёҖеүҜдҪҺеҲҶиҫЁзҺҮеӣҫеғҸиҝӣиЎҢз®ҖеҚ•зҡ„з”»иҙЁжҸҗеҚҮеӨ„зҗҶпјҢе°ҶеӨ„зҗҶеҗҺзҡ„еӣҫеғҸиЎЁзӨәдёә$Y$пјҢеҺҹе§ӢеӣҫеғҸдёә$X$гҖӮ

Path extraction and Representation

иҝҷдёҖеҚ·з§ҜеұӮзҡ„дҪңз”ЁеҸҜд»ҘиЎЁзӨәдёә$\begin{equation}F_{1}\end{equation}$ж“ҚдҪңгҖӮ
$$\begin{equation}F_{1} = max(0, W_{1} * Y + B_{1})\end{equation}\label{eq1}$$
$W_{1}$жҳҜж»ӨжіўеҷЁпјҢ$B_{1}$жҳҜbias. вҖң*вҖқжҳҜеҚ·з§Ҝж“ҚдҪңгҖӮ$W_{1}$еҜ№еә”дёҖдёӘ$n_{1}$дёӘж»ӨжіўеҷЁеӨ§е°ҸжҳҜ$c \times f_{1} \times f_{1}$, cжҳҜйҖҡйҒ“ж•°пјҢ$f_{1}$жҳҜеҚ·з§Ҝж ёзҡ„еӨ§е°ҸгҖӮиҫ“еҮәжҳҜдёҖдёӘ$n_{1}$з»ҙзҡ„зү№еҫҒеӣҫгҖӮ$B_{1}$жҳҜдёҖдёӘ$n_{1}$з»ҙзҡ„еҗ‘йҮҸгҖӮеңЁз»ҸиҝҮжҜҸдёӘж»ӨжіўеҷЁд№ӢеҗҺж·»еҠ ReLU($max(0, x)$)жҝҖжҙ»еҮҪж•°гҖӮ

Non-Linear Mapping

з¬¬дёҖеұӮеҚ·еҹәеұӮд»ҺеӣҫеғҸиЎҘдёҒдёӯжҸҗеҸ–$n_{1}$з»ҙзү№еҫҒгҖӮиҝҷдёҖеҚ·з§ҜеұӮпјҢд»Һ$n_{1}$з»ҙеәҰйқһзәҝжҖ§жҳ е°„еҲ°$n_{2}$з»ҙеәҰгҖӮиҝҷдёҖеҚ·з§ҜеұӮзҡ„дҪңз”Ёзҡ„еҸҜд»ҘиЎЁзӨәдёә$F_{2}$з®—еӯҗгҖӮ
$$\begin{equation}F_{2} = max(0, W_{2} * F_{1}(Y) + B_{2})\end{equation}\label{eq2}$$
$W_{2}$еҜ№еә”дёҖдёӘ$n_{2}$дёӘж»ӨжіўеҷЁеӨ§е°ҸжҳҜ$n_{1} \times f_{2} \times f_{2}$, $n_{1}$жҳҜиҫ“е…ҘйҖҡйҒ“ж•°пјҢ$f_{2}$жҳҜеҚ·з§Ҝж ёзҡ„еӨ§е°ҸгҖӮиҫ“еҮәжҳҜдёҖдёӘ$n_{2}$з»ҙзҡ„зү№еҫҒеӣҫгҖӮ$B_{2}$жҳҜдёҖдёӘ$n_{2}$з»ҙзҡ„еҗ‘йҮҸгҖӮеңЁз»ҸиҝҮжҜҸдёӘж»ӨжіўеҷЁд№ӢеҗҺж·»еҠ ReLU($max(0, x)$)жҝҖжҙ»еҮҪж•°гҖӮ

Reconstruction

иҝҷдёҖйҮҚе»әеҚ·еҹәеұӮзҡ„дҪңз”ЁеҸҜд»Ҙз”Ё$\begin{equation}F_{3}\end{equation}$з®—еӯҗгҖӮ
$$\begin{equation}F_{3} = max(0, W_{3} * F_{2}(Y) + B_{3})\end{equation}\label{eq3}$$
$W_{3}$еҜ№еә”дёҖдёӘcдёӘж»ӨжіўеҷЁеӨ§е°ҸжҳҜ$n_{2} \times f_{3} \times f_{3}$, $n_{2}$жҳҜиҫ“е…ҘйҖҡйҒ“ж•°пјҢ$f_{3}$жҳҜеҚ·з§Ҝж ёзҡ„еӨ§е°ҸгҖӮиҫ“еҮәжҳҜдёҖдёӘcз»ҙзҡ„и¶…еҲҶиҫЁзҺҮйҮҚе»әеӣҫгҖӮ$B_{3}$жҳҜдёҖдёӘcз»ҙзҡ„еҗ‘йҮҸгҖӮ

Training Detail

з«ҜеҜ№з«Ҝжҳ е°„еӯҰд№ еҸҜд»Ҙз”Ё$F$з®—еӯҗиЎЁзӨәгҖӮзҪ‘з»ңеҸӮж•°з”Ё$\theta = {W_{1},W_{2},W_3,B_1,B_2,B_3}$иЎЁзӨәгҖӮ

Loss Function

дҪҝз”ЁMSE(еқҮж–№е·®иҜҜе·®)дҪңдёәжҚҹеӨұеҮҪж•°
$$\begin{equation}L(\theta) = \frac{1}{n} \sum_{i=1}^{n} \parallel F(Y_i;\theta)-X_i \parallel ^2 \end{equation}\label{eq4}$$
nжҳҜbatch_sizeзҡ„еӨ§е°ҸеҚіжҳҜдёҖжү№и®ӯз»ғж ·жң¬зҡ„еӨ§е°ҸгҖӮ

Learning Rate

дјҳеҢ–еҮҪж•°: йҡҸжңәжўҜеәҰдёӢйҷҚпјҲSGDпјү
$$\begin{equation} \Delta_{i+1} = 0.9 \cdot \Delta_{i} + \eta \cdot \frac{\partial L}{\partial W_{i}^{l}}, W_{i+1}^{l} = W_{i}^{l} + \Delta_{i+1} \end{equation}\label{eq5}$$

$l$жҳҜеұӮж•° $l \in {1, 2, 3}$пјҢ iжҳҜиҝӯд»Јж•°, $\eta$жҳҜеӯҰд№ зҺҮгҖӮThe filter weights of each layer are initialized by drawing randomly from a Gaussian distribution with zero mean and standard deviation 0.001 (and 0 for biases). еүҚдёӨеұӮзҡ„$lr=10^{-4}$ ,з¬¬дёүеұӮзҡ„$lr=10^{-5}$.жңҖеҗҺдёҖеұӮдҪҝз”Ёиҫғе°Ҹзҡ„еӯҰд№ зҺҮжҜ”иҫғе®№жҳ“ж”¶ж•ӣгҖӮ
