Overview

　　尽管GANs非常强大，但它很难训练，在实践中经常观察到，基于梯度下降的GAN优化并不会导致收敛。我们的分析表明，具有实例噪声或零中心梯度惩罚的GAN训练是收敛的。令一方面，WGAN和WGAN-GP在有限数次的discriminator的更新中可能不会总是收敛。作者对这些结果进行了讨论，为GAN训练稳定性问题提供了新的解释。通过一个简单而典型的GAN训练实例，分析表明(非正则化)GAN训练并不总是局部收敛的。

  • 通过一个简单的反例证明了(非正则化)梯度下降的GAN优化并不总是局部收敛的
  • 讨论了最近引进的正则化训练技术是否能稳定训练以及如何稳定训练
  • 引入了简化的梯度惩罚，证明了正则化GAN训练动力学的局部收敛性

Super-FAN

发表于 2019-12-30 更新于 2023-09-17 分类于 FH Waline： 本文字数： 2.3k 阅读时长 ≈ 4 分钟

Overview

　　作者提出Sper-FAN: 是一个端到端的能同时完成超分辨率重建和面部landmark的检测。Super-FAN的新颖之处在于，它通过整合一个子网络(FAN)来实现对人脸的对齐，从而将结构信息整合到一个基于gan的超分辨率算法中，并对一种新的热图(heatmap)丢失进行优化(heatmap loss)。举例说明了联合训练这两个网络的好处，不仅在正面图像上，而且在整个面部姿态光谱上，不仅在合成低分辨率(人工降质)图像上，而且在真实世界图像上都取得了良好的结果。

Model

　　Super-FAN包含三个网络，第一个网络是Super-resolution network用于超分辨重建；第二个网络是disciminator用于分辨原图和超分辨重建图片；第三个网络是FAN,用于在超分辨重建图片进行人脸对齐和landmark定位。在测试阶段discriminator是不需要的。

Super-resolution network

　　Super-resolution network网络结构和SRGAN网络结构类似，只不过在细节上有一些改变。

  • Per-block layer distribution. SRGAN网络用了16层在原始低分辨率图片、1层在两倍低分辨率图片，1层在四倍低分辨率图片。标注为16-1-1.由于Super-resolution网络的主要目标是通过幻觉来超级解析其输入，因此仅使用一个更高分辨率的块(如SRGAN)不足以生成清晰的细节，特别是在特别低分辨率的图片。因此Super-FAN采用了12-3-2结构。
  • Building block architecture. Super-FAN试验发现PReLU激活函数并没有比ReLU激活函数试验结果好多少，因此采用ReLU激活函数。
  • On the “long” skip connection.Super-FAN认为分辨率的提高是一个渐进的过程，在此过程中，每一层的表现都要比上一层有所提高，因此，低水平图片的注入对整体表现的影响较小。在实践中，Super-FAN发现使用它的收益非常小。因此不用skip-connection.

Adversarial network

　　发现采用WGAN和使用普通GAN并不会使超分辨率的结果有多大区别。Super-FAN采用WGAN稳定discriminator的训练。Super-FAN的Adversarial network 采用了去除batch normliztion的DCGAN网络。

Face Alignment Network

　　如果只采用上述两个网络进行face　hallucination其实可以生成很好的结果了，但是并没有把人脸结构相关的信息纳入超分辨率过程。由此可能导致超分辨率重建图片姿势和表情不正常。为了解决这一问题，提出通过将heatmap regretion的人脸 landmark 定位网络整合到超分辨率过程中，并优化适当的热图损失(heatmap loss)，来加强低分辨率和高分辨率图像之间的人脸结构一致性。
　　FAN采用heatmap regression去定位landmark: 不是将输出整成68*2的vector，每张图片的landmark被表示成在landmark区域的2D高斯分布，输出在一个通道上。对FAN进行训练，使其回归到2D高斯分布上，也称heatmap(热图)。heatmap包含了形状信息（姿势表情）、空间信息和位置结构联系。FAN的目标就是使得超分辨率重建图片的heatmap和原图heatmap接近。网络使用了FAN带有两个hourglass(参考论文4)

SRCNN

发表于 2019-12-24 更新于 2023-09-17 分类于 SISR Waline： 本文字数： 2.7k 阅读时长 ≈ 5 分钟

Overview

　　SRCNN是深度用在超分辨率重建上的开山之作。其指出传统的基于稀疏矩阵的方法可以看做一个深层神经网络。SRCNN是一个轻量级的网络，只有三个卷积神经网络(CNN)。作者在训练时将图片转化为YCrCb三个通道，只对Y通道进行训练。因为人眼对Y通道比较敏感。

Model

整个SRCNN模型可以解释成三部分:

  • Path extraction and Representation.提取图像Patch，进行卷积提取特征，类似于稀疏编码中的将图像patch映射到低分辨率字典中。
  • Non-linear mapping.将低分辨率的特征映射为高分辨率特征，类似于字典学习中的找到图像patch对应的高分辨字典。
  • Reconstruction.根据高分辨率特征进行图像重建。类似于字典学习中的根据高分辨率字典进行图像重建。

　　首先用双立方插值对预先设计好的尺寸对一副低分辨率图像进行简单的画质提升处理，将处理后的图像表示为，原始图像为。

最近点对问题

发表于 2019-10-19 更新于 2023-09-17 分类于 mathematics Waline： 本文字数： 5.7k 阅读时长 ≈ 9 分钟

考虑确定平面上n个点(x1, y1),(x2,y2)…(xn,yn)集合上的最近点对的问题。

可以很简单的写出暴力法破解：
// Points 为所有点对的集合，using Point = pair<double, double>, 
// a, b 传出参数， 最近点对的两个点。
double get_distance_enforce(const vector<Point> &Points, Point &a, Point &b)
{
    double distance = static_cast<double>(INT32_MAX);
    for (auto i = Points.begin(); i != Points.end(); ++i)
    {
        for (auto j = i + 1; j != Points.end(); ++j)
        {
            double tmp = sqrt(distance_square(*i, *j));
            // 如果跨分割线的两点距离小于已知距离，则记录该距离。
            if (tmp < distance)
            {
                distance = tmp;
                a = *i;
                b = *j;
            }
        }
    }
    return distance;
}
这样得出的算法复杂度为O(n^2).

方法2：可以用递归的方法，分而治之，将一个大问题分解为若干个小问题。

#include <iostream>
#include <random>
#include <chrono>
#include <cmath>
#include <algorithm>
using namespace std;
using Point = pair<double, double>;
constexpr int NO_DISTANCE = 10000;

// 随机生成点对
void set_point(vector<Point> &Points, int size, double x, double y)
{
    std::default_random_engine e(std::chrono::system_clock().now().time_since_epoch().count());
    std::uniform_real_distribution<double> dis(x, y);
    for (auto i = 0; i < size; ++i)
    {
        Points.push_back(Point(dis(e), dis(e)));
    }
}

// 打印点对
void print_points(const vector<Point> &Points)
{
    for (auto i : Points)
    {
        cout << "(" << i.first << ", " << i.second << ")" << endl;
    }
    cout << endl;
}

double distance_square(Point a, Point b)
{
    return (a.first - b.first) * (a.first - b.first) + (a.second - b.second) * (a.second - b.second);
}

// points 所有的点的集合， a, b 传出参数
// Points_X 按X排序的序列， Points_Y 按Y排序的序列。
double closestPoint(const vector<Point> &Points_X, const vector<Point> &Points_Y, Point &a, Point &b)
{
    Point a1, b1, a2, b2;
    int length = Points_X.size();
    double distance;
    if (length < 2)
        return NO_DISTANCE;
    else if (length == 2)
    {
        a = Points_X[0];
        b = Points_X[1];
        distance = sqrt(distance_square(Points_X[0], Points_X[1]));
    }
    else
    {
        vector<Point> pst1(Points_X.begin(), Points_X.begin() + length / 2);
        vector<Point> pst2(Points_X.begin() + length / 2, Points_X.end());
        double d1 = closestPoint(pst1, Points_Y, a1, b1);
        double d2 = closestPoint(pst2, Points_Y, a2, b2);
        if (d1 < d2)
        {
            distance = d1;
            a = a1;
            b = b1;
        }
        else
        {
            distance = d2;
            a = a2;
            b = b2;
        }

        // 一个点在左边 一个点在右边的情况。
        vector<Point> pst3;
        double mid = (Points_X.begin() + length / 2 - 1)->first;
        for (auto point : Points_Y)
        {
            if (abs(point.first - mid) <= distance)
                pst3.push_back(point);
        }

        // 只需要与有序的链接的7个点进行比较就可以了。
        for (auto i = pst3.begin(); i != pst3.end(); ++i)
        {
            for (auto j = i + 1; j != i + 7 && j != pst3.end(); ++j)
            {
                double tmp = sqrt(distance_square(*i, *j));
                // 如果跨分割线的两点距离小于已知距离，则记录该距离。
                if (tmp < distance)
                {
                    distance = tmp;
                    a = *i;
                    b = *j;
                }
            }
        }
    }
    return distance;
}

double get_distance_enforce(const vector<Point> &Points, Point &a, Point &b)
{
    double distance = static_cast<double>(INT32_MAX);
    for (auto i = Points.begin(); i != Points.end(); ++i)
    {
        for (auto j = i + 1; j != Points.end(); ++j)
        {
            double tmp = sqrt(distance_square(*i, *j));
            // 如果跨分割线的两点距离小于已知距离，则记录该距离。
            if (tmp < distance)
            {
                distance = tmp;
                a = *i;
                b = *j;
            }
        }
    }
    return distance;
}
int main()
{
    int points_size;
    cout << "请输入要生成的points的数量 ";
    cin >> points_size;

    cout << "********************************************************************************" << endl;
    cout << "随机生成的点是" << endl;
    vector<Point> Points;
    set_point(Points, points_size, -100.0, 100.0);
    print_points(Points);

    auto start1 = std::chrono::system_clock::now();
    vector<Point> Points_X(Points);
    vector<Point> Points_Y(Points);

    sort(Points_X.begin(), Points_X.end(), [](Point a, Point b) { return a.first < b.first; });
    sort(Points_Y.begin(), Points_Y.end(), [](Point a, Point b) { return a.second < b.second; });

         << "秒" << endl;

    auto end2 = std::chrono::system_clock::now();

}

数学归纳法和强归纳法

发表于 2019-10-16 更新于 2023-09-17 分类于 mathematics Waline： 本文字数： 1.1k 阅读时长 ≈ 2 分钟 
数学归纳法的原理： 为证明对所有的正整数n，p(n)为真，其中p(n)是一个命题函数
需要完成两个步骤
    基础步骤： 证明命题p(1)成立
    归纳步骤： 证明对每个正整数k来说， p(k) --> p(k+1)

    强归纳法： 要证明对所有的正整数n而言，都有p(n)为真，其中p(n)为命题函数，我们
要完成如下两个步骤：
    基础步骤： 证明p(1)为真
    归纳步骤： 要证明对所有正整数k来说，蕴含式 p(1)&p(2)...p(k)  --> p(k+1)也为真

example：
a) Determine which amounts of postage can be formed
using just 4-cent and 11-cent stamps.
b) Prove your answer to (a) using the principle of math-
ematical induction. Be sure to state explicitly your
inductive hypothesis in the inductive step.
c) Prove your answer to (a) using strong induction. How
does the inductive hypothesis in this proof differ from
that in the inductive hypothesis for a proof using math-
ematical induction?

a)、4,8,11,12,15,16,19,20,22,23,24,26,27,28，以及大于等于30的所有值
b)、令P(n)是命题：我们可以只用4分邮票和11分邮票组成n分邮资。证明：对于所有 n >= 30, P(n)为真。
基础步骤：可以组成k分邮资，如何组成k+1分邮资
归纳步骤：如果k分邮资中有一张11分邮票，可以用3张4分邮资代替组成k+1；否则k分邮资只由4分邮票构成。因为k>=30,可以用三张11分邮票代替8张4分邮票，组成k+1分邮资。
c)P(n)与b中相同。
基础步骤：P(30) = 11+11+4, p(31) = 11+4+4+4+4+4, p(32)=4+4+4+4+4+4+4+4, p(33) = 11+11+11;
归纳步骤：假设归纳假设p(j)对于所有j(30<=j<=k)都为真，这里k是大于等于33的任意整数。我们要证明p(k+1)为真。因为k-3>=30,所以p(k-3)为真，即可以组成k-3分邮资。在这个信封上在贴一张4分邮票，就组成了k+1分邮资。在这个证明中，我们的归纳假设是p(j)对所有介于30和k之间的值都为真，而不只是p(30)为真。

#include <iostream>
#include <vector>

// 找出列表中的第一只单身狗
int find_single_dog(const vector<int> &match_for_man)
{
    for(auto i=0; i<match_for_man.size(); ++i)
    {
        if(match_for_man[i] == -1)
            return i;
    }
    return -1;
}

// 比较自己和原配的排名 看看能否成功挖墙脚
bool check_rank(const vector<int> &preference_of_women, int man_index, int old_man_index)
{
    for(auto i:preference_of_women)
    {
        if(i == man_index)
            return true;
        else if(i == old_man_index)
            return false;
    }
    return false;

}

// 打印结果
void print_res(const vector<int>& match_for_man, const vector<int>& match_for_woman)
{
    for(auto i=0; i<match_for_man.size(); ++i)
    {
        cout << "男生" << i << "号 --->" << " 女生" << match_for_man[i] << "号" << endl;
    }
    for(auto i=0; i<match_for_woman.size(); ++i)
    {
        cout << "女生" << i << "号 --->" << " 男生" << match_for_woman[i] << "号" << endl;
    }
}
int main()
{
    // 男生的喜好列表
    vector<vector<int>> preference_of_man{vector<int>{0, 1, 3, 2}, vector<int>{0, 2, 3, 1}, vector<int>{3, 0, 2, 1}, vector<int>{0, 3, 1, 2}};
    // 女生的喜好列表
    vector<vector<int>> preference_of_woman{vector<int>{3, 0, 1, 2}, vector<int>{3, 2, 1, 0}, vector<int>{3, 2, 0, 1}, vector<int>{2, 3, 0, 1}};
    int length = preference_of_man.size();
    vector<int> match_for_man(length, -1);
    vector<int> match_for_woman(length, -1);
    // vector<int> matched_len(length, -1);
    int man_index;
    int woman_index;
    // 循环查找列表中的单身狗
    while((man_index = find_single_dog(match_for_man)) > -1)
    {
        print_vec(match_for_man);
        for(int index=0; index < length; ++index)
        {
            // 找出该单身狗所心仪的对象
            if(match_for_woman[woman_index] == -1)
            {
                match_for_man[man_index] = woman_index;
                match_for_woman[woman_index] = man_index;
                break;
            }
            // 很不幸 该对象已经有伴了。。。 看一下能不能挖走墙角
            else if(check_rank(preference_of_woman[woman_index], man_index, match_for_woman[woman_index]))
            {
                // 如果能挖走墙角
                // 踢掉原配
                match_for_man[match_for_woman[woman_index]] = -1;
                match_for_woman[woman_index] = man_index;
                match_for_man[man_index] = woman_index;
                break;
            }
        }
    }

    print_res(match_for_man, match_for_woman);
    return 0;
}

背包问题的解决

int knapsack(int weight, const vector<int>& things)
{
    vector<int> dp(weight+1);
    for(auto i:things)
    {
        // 初始化背包数组
        dp[i] = i;
    }
    for(auto i=1; i<=weight; ++i)
    {
        // 跳过哪些已经被赋值的元素
        if(dp[i])
            continue;
        vector<int> tmp;
        for(auto thing:things) 
        {
            if(i-thing > 0)
                // 添加可能的值进入dp数组
                tmp.push_back(dp[i-thing] + thing);
        }
        if(tmp.size())
            // 递推计算可能的最大的值
            dp[i] = *max_element(tmp.cbegin(), tmp.cend());
    }
    return dp[weight];

}
