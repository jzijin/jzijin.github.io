SiGAN

Posted on 2020-01-08 In FH

Overview

　　虽然GAN可以产生逼真的人脸图像，但是会产生身份信息的丢失。提出SiGAN解决这个问题。SiGAN网络最上层有两个generator和一个discriminator。将重建误差和身份标签信息以成对的方式合并到SiGAN的损失函数中。现在大多数的方法都只注重重建质量而不注重身份信息，SiGAN注重身份信息的恢复。

  • 提出SiGAN可以恢复高质量的图片并且保持身份信息不丢失
  • 在不需要真标签的情况下，利用SiGAN网络嵌入弱一对标签信息，极大地降低了标签成本，提高了针对不可见身份的人脸的方法的可扩展性
  • 感知和定量实验表明，该算法具有良好的性能和较强的泛化能力。

Model

　　SiGAN有两个generator和一个discriminator。在这一对Generator中，一对LR图片作为先验信息指导HR图片的生成。具体地说，给定一对LR面以及一个二元两两标识指示器，指示这两个LR面是否属于同一标识，设计了contrastive loss，用于将二元两两标识标签嵌入到生成器对中进行训练。reconstruction loss 为重建图片和原图之间的L1距离。GAN loss定义为$L_{GAN} = logD(x) + log(1-D(G(z)))$。整个SiGAN的loss定义为$L_{SiGAN} = L_{GAN} + L_{c} + L_{r}$

　　Generator是一个SRCNN。在Generator中键入两个上采样网络放大4倍。Discriminator和DCGAN类似。
