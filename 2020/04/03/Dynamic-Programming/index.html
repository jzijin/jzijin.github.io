动态规划

动态规划与分治方法相似，都是通过组合子问题的解来求解原问题。动态规划方法通常用来求解最优化问题。这类问题可以有喝多可行解，每个解都有一个值，我们希望寻找具有最优值的解。我们通常按如下4个步骤来设计一个动态规划算法:

  1. 刻画一个最优解的结构特征。
  2. 递归地定义最优解的值。
  3. 计算最优解的值，通常采用自底向上的方法。
  4. 利用计算出的信息构造一个最优解。
