RaGAN:The relativistic discriminator: a key element missing from standard GAN

发表于 2020-02-18 更新于 2023-09-17 分类于 GAN 阅读次数： Waline： 本文字数： 987 阅读时长 ≈ 2 分钟

Overview

　　在传统的GAN（SGAN）中，判别器D估计输入数据是真实数据的概率。生成器G训练增加让判别器识别生成数据为真实数据的概率。我们认为生成器也应该同时降低让判别器识别真实数据是真实的概率，因为，第一，在一个mini-batch中的数据都是假的，第二，这样能使得散度最小化。第三，在最优设置下，SGAN等价于积分概率度量(IPM) GANs。

　　结果表明，利用相对论鉴别器估计真实数据比随机采样的假数据更真实的概率效果更好。我们还提出了一个变种，其中鉴别器估计的概率，给定的真实数据比虚假数据平均而言更真实。我们将这两种方法推广到非标准GAN损失函数，并分别将它们称为相对论性GAN(RGANs)和相对论性平均GAN(RaGANs)。基于积分概率指标的GAN是RGANs的一个子集。
　　我们观察到,第一,RGANs和RaGANs相比与他们的非相对版本能生成高质量的数据样本而且更稳定，第二，带有梯度惩罚的标准RaGAN生成的数据比WGAN-GP的质量更好，而每个生成器更新只需要一个鉴别器更新(将达到最新水平所需的时间减少了400%)，第三，RaGANs能够从一个非常小的样本(N=2011)生成貌似合理的高分辨率图像(256x256)，而GAN和LSGAN不能;这些图像的质量明显优于WGAN-GP和SGAN的光谱归一化处理。
